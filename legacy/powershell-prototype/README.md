# PowerShell Prototype Archive

This folder contains the previous prototype implementation.

It is preserved only for historical reference and migration of product ideas. It is not part of the professional product runtime.

Do not use this folder to:

- install Direnix in a VM;
- validate the product architecture;
- create a Windows Service;
- build release artifacts;
- document current customer deployment.

The active product code and installer live under:

```text
product/
product/installer/Direnix.Msi
```

The active MSI artifact is:

```text
product/installer/Direnix.Msi/bin/x64/Release/Direnix.msi
```
